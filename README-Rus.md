# Tab+ · [English](./README.md)

Расширение для [Spicetify](https://spicetify.app), которое устраняет зависание интерфейса Spotify при удержании клавиши **Tab**.

![Превью](preview.png)

---

## Проблема

При удержании Tab браузерный движок Chromium (на котором работает десктопный Spotify) генерирует **30–50 событий `keydown` в секунду**. Каждое из них запускает систему управления фокусом Spotify, которая перестраивает дерево доступности React для нового элемента. При такой частоте поток JS переполняется — интерфейс полностью зависает.

## Решение

Tab+ перехватывает события клавиатуры в **фазе захвата** (раньше любых обработчиков Spotify и раньше стандартного действия браузера). Для авто-повторных событий (`e.repeat === true`) расширение работает в одном из двух режимов:

- **Троттлинг** — разрешает один сдвиг фокуса за настраиваемый интервал (по умолчанию 225 мс)
- **Блокировка** — подавляет все повторы; фокус двигается только при отдельных нажатиях

Одиночные нажатия никогда не затрагиваются.

---

## Возможности

- Моментальное устранение зависания при удержании Tab — без перезапуска Spotify после изменения настроек
- Три пресета под мощность ПК: **Слабый** (350 мс), **Средний** (225 мс), **Мощный** (100 мс)
- Точная настройка интервала ползунком (50–1500 мс) с прокруткой колёсиком мыши
  - Зажмите **Alt** при прокрутке или перетаскивании для шага **1 мс**
- Опциональный троттлинг стрелок для списков и сеток (внутри текстовых полей не работает)
- Режим блокировки для максимальной защиты на очень слабом железе
- Окно настроек доступно из меню профиля Spotify
- Конфигурация сохраняется в `localStorage` — переживает перезапуски Spotify

## Установка

### Через маркетплейс Spicetify *(рекомендуется)*

1. Откройте Spotify → меню профиля → **Marketplace**
2. Найдите **Tab+**
3. Нажмите **Установить**

### Вручную

1. Скачайте `tabplus.js`
2. Скопируйте в папку расширений Spicetify:
   - **Windows:** `%APPDATA%\spicetify\Extensions\`
   - **macOS/Linux:** `~/.config/spicetify/Extensions/`
3. Выполните:
   ```
   spicetify config extensions tabplus.js
   spicetify apply
   ```

## Использование

Откройте Spotify → нажмите на **фото профиля** → **Tab+ Settings**

| Настройка | Описание |
|---|---|
| **Включено** | Главный переключатель |
| **Режим** | Троттлинг (замедлить) или Блокировка (не двигать при удержании) |
| **Интервал** | Пауза в мс между разрешёнными сменами фокуса (только в режиме троттлинга) |
| **Стрелки тоже** | Применить ту же защиту к удержанию стрелок в списках и сетках |

### Пресеты

| Пресет | Интервал | Стрелки |
|---|---|---|
| Слабый ПК | 350 мс | ✅ |
| Средний ПК | 225 мс | ❌ |
| Мощный ПК | 100 мс | ❌ |
